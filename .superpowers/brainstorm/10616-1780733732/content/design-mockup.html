Design Details

Map Features

  • GeoJSON country boundaries with choropleth coloring
  • Sequential color scale: light blue → orange → red
  • Hover tooltip showing country name, IP count, issue count
  • Click to navigate to assets filtered by country
  • Legend showing color scale and IP range

Table Features

  • Country name, IPs with issues (with progress bar), Total IPs
  • Click row to highlight country on map
  • Sort by IP count or country name
  • Footer with totals
  • Responsive design for different screen sizes
